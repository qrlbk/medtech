# Демо-сценарий (1–3 минуты)

Предусловие: `make up && make migrate && make seed` (или см. README).

1. **Поиск.** Открыть http://localhost:3000, ввести «ОАК» → выбрать «Общий анализ
   крови (ОАК)» из автодополнения. Показать список клиник с ценами, бейдж
   «Лучшая цена», дату обновления и ссылку на источник.
2. **Фильтры и сортировка.** Выбрать город «Алматы», сортировку «по расстоянию»
   (разрешить геолокацию) и «Онлайн-запись».
3. **Аналитика.** Показать панель min/avg/max и число клиник.
4. **Карта.** Показать клиники на карте справа (Leaflet).
5. **Сравнение.** Отметить 2–3 клиники чекбоксом «сравнить» → таблица сравнения.
6. **Карточка клиники + история.** Кликнуть на клинику → раскрыть услугу → график
   истории цены.
7. **Аналитика рынка (API).** `GET /api/v1/analytics/market` — услуги с самым большим
   разбросом цен (где экономия максимальна).
8. **Админка.** http://localhost:3000/admin (admin@medprice.kz / admin) → очередь
   ручной разметки → привязать непривязанную услугу → она исчезает из очереди и
   запоминается как синоним.
9. **Ручной запуск парсинга.** Кнопка «Запустить парсинг» в админке.
10. **Мониторинг.** Grafana http://localhost:3001 — свежесть данных, unmatched,
    запуски парсеров, латентность поиска.
